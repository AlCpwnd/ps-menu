# PS-Menu
Simple module to generate interactive console menus (like yeoman)

# Examples:

```powershell
Menu @("option1", "option2", "option3")
```
## Installing PS-Menu

You can install it from the PowerShellGallery using PowerShellGet

```powershell
Install-Module PS-Menu
```
